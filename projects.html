Poseidon's Eye

We propose a cloud-based information system for the dynamic, secure and user-friendly information system for natural resource and infrastructural data for the Canadian coastal zone. Our system is a utilization of four distinct information systems technologies to provide a robust and dynamic solution for this challenge. These four components are namely; the Relational Database , the Data Management interface, the Modelling interface and, the User Interface . This proposal will reference these features as such. These four systems and their underlying functionality will come together to provide the desired services.
